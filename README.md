+# Skidloggen
+
+En mobilanpassad skidlogg för längdskidåkning där du kan logga:
+
+- Klassiskt
+- Skejt
+- Rullskidor
+- Stakmaskin
+
+## Funktioner
+
+- Logga datum, distans och tid för alla stilar.
+- Höjdmeter och Stifa används för Klassiskt, Skejt och Rullskidor.
+- För Stakmaskin används inte höjdmeter/Stifa och de räknas inte med i sammanfattningarna.
+- Beräknar kilometertempo.
+- Automatisk säsongshantering (juni till juni, t.ex. 25/26).
+- Sammanfattning med totaler för aktuell säsong.
+- Summering per stil.
+- Cirkeldiagram över distans per stil.
+- Ändra och ta bort pass.
+
+## Kör lokalt
+
+Öppna `index.html` i webbläsaren.
